Why it matters

job Detect any Claude Code session that has ever run claude-fable-5 and pin it back the moment it drifts to another model. proof The match is history-based, not latest-model-only, so a session stays watched even after it has already drifted.
job Bind a session to a pane by the Herdr-reported session id, never by focus or a cwd guess. proof Ambiguous multi-pane matches hard-block armed remediation instead of guessing which pane owns the session.
job Act while the downgraded turn is still running, not after it finishes on the wrong model. proof The fast path queues /compact mid-turn and sends /model on the next idle pass — a dialog-free switch, no cache-rewind confirmation.
job Stay opt-in and hands-off deliberate sessions. proof Sessions that have never run Fable and match no configured target are ignored, including deliberate Sonnet or Opus work.

How it watches

One Rust binary, three boundaries: Claude transcript JSONLs, live Herdr pane state, and an optional config file. No background daemon beyond the LaunchAgent that runs the armed watch on a schedule.

detect counterspell status discovers recent transcripts, maps them to Herdr panes by cwd, and shows watched, unmatched, and live-pane rows without a recent transcript. gate Armed remediation only fires once transcript-quiet, pane-idle, and debounce all pass — or the session-id fast path is bound and the pane is working.
remediate counterspell watch --arm sends a plain /compact handoff, waits for idle, then sends /model claude-fable-5. observe counterspell ui serves a local Herdr column drilldown — workspace, pane, session, policy — with a menu-bar dot via the bundled SwiftBar plugin.
