What's new in Counterspell

These notes say what each release lets you do, in plain terms. Counterspell has one tagged release, v0.1.0, and has kept shipping since.

2026-07-07 - recovery

Recover a session instantly after a restart or /clear

When a session gets reset, you no longer wait for the next automatic check to protect it again — re-attach it yourself in one command.

  • counterspell rebind re-attaches a pane to its session right after a restart, a resume, or a /clear, instead of waiting for the next automatic check.
  • When it cannot tell for sure which pane owns a session, it now holds off rather than guessing — and quietly recovers on its own once a pane reports in.

Shipped and verified live.

2026-07-04 - mid-turn fix

A downgrade now gets fixed before your turn even finishes

You stop losing a whole turn to the wrong model — the moment a session drifts, the fix is already on its way.

  • The fix lands mid-turn: the instant a session drifts, Counterspell queues a /compact handoff and sends /model on the next idle pass — no more watching a wrong-model turn run to the end.
  • It always acts on the right pane, matched by the Herdr-reported session id rather than a guess.
  • The whole thing now installs, sets up its dashboard, and starts watching from a single command: counterspell setup --install-ui --load-ui.
2026-07-03 - dashboard

See everything it is watching in one place

You get a clear, local view of every session and policy — and safer behavior when two panes look alike.

  • A local dashboard now shows every pane, session, and the policy applied to each, so you can see exactly what Counterspell is doing.
  • When two panes look like the same session, it only picks one if the answer is unambiguous — otherwise it waits.
  • Signed builds for both Apple-silicon and Intel Macs, so install just works.
2026-07-02 v0.1.0

Start protecting your Fable sessions

The first release: point Counterspell at your machine and it finds your Claude Code sessions, shows which ones it is watching, and never touches anything you did not ask it to.

  • One command finds your recent Claude Code sessions, maps them to their Herdr panes, and tells you which are watched and which are ignored.
  • Opt-in from the very first run — a session is only ever changed if it matches a target you set, never by guess.

Tagged as v0.1.0 with signed download archives.
