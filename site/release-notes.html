What's new in Counterspell

v0.1.0 2026-07-02 — present

Everything shipped so far, in plain terms: point Counterspell at your machine and it finds your Claude Code sessions, keeps the ones you chose to run on Fable pinned there, and stays out of the way of everything else.

  • Watches every Claude Code session that has ever run Fable and puts it back automatically the moment it drifts to another model — mid-turn, before you lose the whole turn to the wrong model.
  • Binds each session to its own terminal window by a stable session id, never by which window happens to be focused, and holds off instead of guessing when a match is ambiguous.
  • Re-attaches a window to its session yourself, in one command, right after a restart, a resume, or a /clear — no waiting for the next automatic check.
  • A global switch turns every enforcement action off in one command, and shows you whether the background watch has picked up the change.
  • A local dashboard lists every window, session, and policy at a glance, fed by its own dedicated event stream, with a menu-bar indicator always showing what it's watching.
  • A native desktop app ships alongside the command line, with a tray icon and window controls for anyone who prefers a GUI to a terminal.
  • Installs from a single command, with signed release binaries for both Apple-silicon and Intel Macs, and a public marketing site and docs alongside the binary.

Tagged as v0.1.0 with signed download archives.
