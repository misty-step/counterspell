Get started

Counterspell is one Rust binary and a local config file. Everything below is copied from the shipped README and ARCHITECTURE docs and checked against the CLI's own --help output — run it from a checkout or a release download, not from this page.

Install

Download a prebuilt macOS binary from the latest release:

curl -LO https://github.com/misty-step/counterspell/releases/latest/download/counterspell-<tag>-aarch64-apple-darwin.tar.gz
curl -LO https://github.com/misty-step/counterspell/releases/latest/download/counterspell-<tag>-aarch64-apple-darwin.tar.gz.sha256
shasum -a 256 -c counterspell-<tag>-aarch64-apple-darwin.tar.gz.sha256
tar xzf counterspell-<tag>-aarch64-apple-darwin.tar.gz
install counterspell-<tag>-aarch64-apple-darwin/counterspell /usr/local/bin/counterspell

Use the x86_64-apple-darwin archive on Intel Macs, or build from source: 

cargo install --path .

Run guided setup

Setup creates the config if needed, installs the SwiftBar plugin, and loads the Herdr annotation and armed-watch LaunchAgents: 

counterspell setup --install-ui --load-ui

The armed watch LaunchAgent runs counterspell watch --arm every 60 seconds from then on. Reach for a manual counterspell watch --arm only when you want to force a pass immediately.

Check what it sees

counterspell doctor
counterspell status
counterspell status --json

doctor checks the local install, config, Herdr reachability, and whether the armed-watch LaunchAgent is actually scheduled. status lists recent Claude transcripts, the Herdr panes they map to, and whether each one is watched.

Open the dashboard

counterspell ui

Serves a local Herdr column drilldown on 127.0.0.1 — workspace, pane, session, policy — and opens it by default. Use counterspell ui --no-open to start the server without launching a browser.

Add extra coverage

Any session that has ever run claude-fable-5 is watched automatically. To watch a session or project that hasn't run Fable yet, add an explicit target: 

counterspell target add --session-id db72af91-c78f-4b3f-80be-6dca7c264f75
counterspell target list

target add defaults to claude-fable-5; pass --target-model only when the extra coverage should enforce a different model.

Re-bind a pane after restart or clear

Herdr learns a pane's session id from Claude's SessionStart hook. After a restart, resume, or /clear, re-assert the binding without waiting for the next hook fire: 

counterspell rebind --verify

Verify a build is actually working

cargo fmt -- --check && cargo test && cargo clippy --all-targets -- -D warnings
cargo install --path .
counterspell doctor
counterspell status
counterspell status --json
counterspell watch --arm
