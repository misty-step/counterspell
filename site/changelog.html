Release notes

Counterspell has one tagged release, v0.1.0, and kept shipping past it. These are the milestones so far, written for someone deciding whether to trust it near a live Fable session — each entry links to the merged proof.

2026-07-07 - pane rebind

Re-bind a pane without waiting for the next hook

  • counterspell rebind re-asserts a pane's Herdr session binding after a restart, resume, or /clear — no need to wait for Claude's next SessionStart event.
  • Ambiguous cwd-based session discovery is refused rather than guessed, and Herdr self-heals when no Claude pane reports an agent_session at all.

Evidence: merged, redeployed, and verified live.

2026-07-04 - fast path + auto-Fable

Act while the downgraded turn is still running

  • Sessions bind to panes by the Herdr-reported session id, never by focus or a cwd guess — the safety property the rest of the design depends on.
  • The fast path queues a mid-turn /compact the moment a session drifts, then switches model on the next idle pass — a dialog-free switch instead of waiting for the whole turn to finish on the wrong model.
  • The automatic Fable target now takes precedence over configured targets, and counterspell setup --install-ui --load-ui productizes the whole install into one command.

2026-07-03 - dashboard + hardening

A visible dashboard, a safer tiebreak

  • The local dashboard became a Herdr column drilldown — workspace, pane, session, policy — and multi-pane cwd matches now resolve to the focused pane only when exactly one pane is focused, staying ambiguous otherwise.
  • Tagged builds publish signed macOS binaries for both architectures automatically, and the dashboard's mutating routes are CSRF-guarded.

2026-07-02 - v0.1.0

First cut: watch, status, opt-in

  • One Rust CLI that discovers recent Claude transcripts, maps them to live Herdr panes, and reports watched vs. ignored sessions.
  • Opt-in from the start — a session is only ever remediated if it matches an explicit or automatic target, never guessed.

Evidence: v0.1.0 tagged and published with signed release archives.
