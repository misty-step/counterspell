Ship Counterspell as a small Rust CLI.

The work is done when counterspell watch records the current Codex session and cwd, and counterspell status shows watched sessions mapped to Herdr panes by running herdr pane list and matching cwd.

Chosen Design

Rust binary, global state file, Herdr as an external command. The CLI owns durable watch state; Herdr remains the live pane source of truth.

Standards

Small public surface, path-normalized cwd matching, no hidden repo state, clear text output.

Acceptance

counterspell watch and counterspell status run locally after install.

Verification

CLI integration tests plus live command transcripts against the current Herdr pane list.

Stop If

Herdr JSON shape contradicts the addendum, or no session/cwd can be determined.

Implementation

  • Document the command contract.
  • Add failing integration tests for watch/status and cwd matching.
  • Implement state load/save, session detection, Herdr parsing, and table rendering.
  • Install locally and run the real commands.

Quality System

  • Gate: cargo fmt -- --check && cargo test && cargo clippy --all-targets -- -D warnings.
  • Fresh critic must check that Herdr failures are not hidden and session ids are not invented.
  • Residual risk to report: exact full brief was not present in the checkout; addendum is treated as source of truth.
